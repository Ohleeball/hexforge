## Genre Reference

- **Vampire Survivors** — action loop, upgrade prompt UX, horde escalation
- **Dome Keeper** — player-only defense, risk/reward of leaving base, resource runs
- **Teamfight Tactics** — economy system, interest mechanic, triple-upgrade leveling, synergies
